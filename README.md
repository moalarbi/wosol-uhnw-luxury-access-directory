# WOSOL UHNW Luxury Access Directory

Static internal research hub for global UHNW luxury access providers across private aviation, superyachts, luxury concierge, bespoke travel, private clubs, and GCC / Saudi-relevant services.

## Files

- `index.html` - page structure and executive research sections.
- `style.css` - premium dark WOSOL styling.
- `script.js` - company dataset, filters, search, modal, sortable table, sources, print, and back-to-top interactions.

## Open Locally

Open `index.html` directly in a browser. No build step, server, React, Tailwind, or external framework is required.

## Updating Companies

Edit the `companies` array in `script.js`. Each company record controls the card, modal, comparison table, and source list automatically.

Use `Not publicly stated` when a field is not confirmed from public sources.
